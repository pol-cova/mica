# Security Policy

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability. Use GitHub’s private security advisory flow for this repository, or contact the maintainer privately with a minimal reproduction and impact summary.

## Scope

Mica is designed for read-only telemetry access. Never include credentials, webhook URLs, customer data, or production payloads in issues, pull requests, skills, or examples.

## Supported version

The `main` branch is the supported development version.
